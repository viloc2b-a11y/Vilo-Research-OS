# Pharmacy Coordinator Challenge Report

## 1. Challenge Mode Scope
The VIP engine generated and answered 75 simulated Q&A scenarios strictly related to Pharmacy Runtime Operations. Below is an excerpt of the most critical operational queries.

## 2. Extracted Challenge Q&A

**Q: Refrigerator reached 10°C overnight. What happens?**
*A:* This is a Temperature Excursion. The IP must be immediately moved to a compliant quarantine space. Dispensing from this lot must halt. Sponsor must be notified within 24h, and IP cannot be used until a formal release form is provided by the sponsor.

**Q: Drug shipment arrives damaged. What is required?**
*A:* Do not place in active inventory. Quarantine the shipment. Photograph the damage, download the transit temperature logger, and notify the IRT/Sponsor immediately for a replacement shipment.

**Q: Temperature logger failed. What is required?**
*A:* Immediately check the backup min/max thermometer to establish the boundary of the failure period. Quarantine the IP. Notify the Sponsor. The IP remains in quarantine until the sponsor reviews the backup data and approves release.

**Q: IP count does not reconcile (Missing vial). What is required?**
*A:* Do not adjust logs arbitrarily. Document the missing vial, write an explanatory note to file (e.g., dropped/shattered, patient lost bottle), and notify the Clinical Research Associate (CRA)/Sponsor.

**Q: Subject returns partial medication. What is required?**
*A:* The unblinded pharmacist or CRC must count the remaining pills/measure the remaining volume, record it on the Subject Accountability Log, and store the return in a secure "Returns" area pending CRA verification.

**Q: Can dispensing continue during quarantine?**
*A:* Absolutely not for the affected lot. Only unaffected lots (e.g., stored in a separate compliant fridge) can be dispensed.

**Q: Who must be notified during a dispensing error (Wrong IP to Wrong Subject)?**
*A:* The Principal Investigator (PI) immediately for safety monitoring, followed by the Sponsor and the IRB/Ethics Committee as this constitutes a Major/Critical Deviation.

**Q: What creates a critical pharmacy audit finding?**
*A:* Dispensing IP that has expired, dispensing IP that suffered an unreported temperature excursion, or unblinding the study team due to careless document storage.
