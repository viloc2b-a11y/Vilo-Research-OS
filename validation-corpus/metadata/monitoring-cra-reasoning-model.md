# CRA Reasoning Model (The Auditor Brain)

## 1. How a Senior CRA Thinks
A Coordinator thinks: *"I need to complete this visit."*
A CRA thinks: *"If the FDA inspects this visit in 3 years, can they prove it actually happened exactly as documented?"*

The CRA operates on the presumption of doubt. Without ALCOA+ (Attributable, Legible, Contemporaneous, Original, Accurate), the data is presumed false.

## 2. Finding Triggers & Adjudication

### Trigger: Missing Timestamp on Source
- **CRA Thought:** "I cannot prove when this happened. I cannot verify protocol windows. Is this a systemic issue or an isolated slip?"
- **Action:** Issue Query. If unresolved, log Minor/Major Deviation based on the endpoint's criticality.

### Trigger: Late Source Entry (Documented days later)
- **CRA Thought:** "Human memory is fallible. This data is no longer 'Contemporaneous'. I must scrutinize the accuracy."
- **Action:** Request a Note to File (NTF) explaining the late entry. Escalate if it becomes a pattern.

### Trigger: Unresolved Query > 30 Days
- **CRA Thought:** "The site is ignoring data cleaning. The PI lacks oversight of the data management process. Database Lock is at risk."
- **Action:** Elevate to Site PI. Issue a formal "Action Item" in the follow-up letter.

### Trigger: Missing PI Signature on Lab Review
- **CRA Thought:** "The PI did not review this safety data. A Clinically Significant (CS) lab might have been missed. The patient was placed at risk."
- **Action:** Major Finding. Escalate immediately. Force PI to review all backlog.

### Trigger: Eligibility Criterion Undocumented
- **CRA Thought:** "The patient might not belong in this study. Every safety event and efficacy data point is now compromised."
- **Action:** Critical Finding. Halt dosing until eligibility is retroactively proven. If unprovable, patient is withdrawn.

## 3. Escalation Vectors
- **To CTM (Clinical Trial Manager):** Trends of poor performance, unresponsiveness.
- **To Medical Monitor:** Safety concerns, missed SAEs, eligibility doubts.
- **To Sponsor QA / Compliance:** Fraud suspicions, falsified signatures, critical ALCOA violations.
