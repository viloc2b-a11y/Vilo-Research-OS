# PI Oversight Mapping

This matrix maps critical clinical procedures to the legally required oversight authority within the Vilo OS Medical Authority Engine.

| Procedure | Requires PI | Allows SI | Requires Medical Monitor | Requires Sponsor | Requires Central Lab |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Lab Review (CS/NCS)** | NO | YES | NO | NO | NO |
| **Eligibility** | YES | NO (Final signature by PI) | NO | NO | NO |
| **SAE Causality** | YES | NO | YES | YES | NO |
| **Dose Modification** | YES | NO | NO | NO | NO |
| **Protocol Waiver** | YES | NO | YES | YES | NO |
| **Pregnancy Result** | YES | YES | NO | NO | NO |
| **Vitals Review** | NO | YES | NO | NO | NO |
| **Unblinded Inventory**| NO (Unblinded Pharm) | NO | NO | NO | NO |

## Principles
1. **SI (Sub-Investigator) Scope:** SIs may conduct routine medical reviews (Labs, Vitals, ECGs) but cannot execute final determinations of Eligibility or SAE causality.
2. **Medical Monitor Escalation:** Required for SAEs and Protocol Waivers to ensure Sponsor oversight.
3. **Blinding Exception:** PIs and SIs are intentionally excluded from unblinded pharmacy workflows to preserve trial integrity.
